# waymo-perception-pipeline
End-to-end autonomous driving perception pipeline using Waymo Open Dataset — object detection, multi-object tracking, trajectory extraction, and motion prediction

-  End-to-end autonomous driving perception pipeline on Waymo Open Dataset
- Parses raw TFRecord sensor data from Google Cloud Storage
- Visualizes ground-truth bounding boxes across 5 synchronized cameras
- Multi-object tracking using Waymo's persistent ground-truth IDs
- Trajectory extraction and motion prediction with velocity features
